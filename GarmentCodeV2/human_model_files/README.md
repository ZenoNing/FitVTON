# SMPL / SMPL-X model files

Body model weights are **not** shipped with this repository. Download them from the official sites after registration:

- SMPL: https://smpl.is.tue.mpg.de/
- SMPL-X: https://smpl-x.is.tue.mpg.de/

See [GarmentCodeV2 and Local Simulation Data](../../docs/garmentcodev2.md#smpl--smpl-x-body-models) for filenames and placement.
